## Acerca del proyecto

Este proyecto consiste en una pequeña aplicación web capaz de obtener información de difusiones RSS y posteriormente presentarlas de manera cómoda para la vista, entre sus funcionalidades se encuentra:

- Registro de nuevos enlaces desde los cuales obtener difusiones RSS.
- Almacenamiento en la base de datos la información obtenida de los enlaces.
- Filtrado de información de la base de datos en función de su fecha de emisión.
- Buqueda de información mediante palabras clave en título, descripción o autor.
- Almacenamiento de la información obtenida en documentos de texto / hojas de cálculo (Trabajo en proceso).
- Descarga de dichos documentos de texto / hojas de cálculo (Trabajo en proceso).


## Información adicional

- Se incluye la copia de una base de datos muestra en el directorio dbBackup.
